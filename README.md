# Unit Utils

A simple unit converter app. The main goal of this project was to publish my first app on the Play Store 🚀

## Overview

Unit Utils is a lightweight app that performs unit and currency conversions directly on your device. It’s designed to be fast, simple, and private.

## Features

- Fast, local unit conversions ⚡  
- Currency conversion with live exchange rates  
- No accounts, no tracking  
- No data storage  

## Privacy

Unit Utils does not collect, store, or share any personal data 🔒

- No analytics  
- No ads  
- No user accounts  

Currency exchange rates are fetched from https://frankfurter.dev. No personal data is intentionally sent.

## Data Handling

- All calculations run locally  
- No data is stored between sessions  
- Exchange rates are kept in memory only while the app is open
